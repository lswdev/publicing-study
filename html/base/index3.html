헤더 영역 : 문서나 특정 섹션의 header를 정의할 때 사용
네비게이션 영역 : 네비게이션 링크의 집합을 정의할 때 사용
메인 영역 : 문서의 중심 주제 또는 주요 기능과 직접적으로 관련되어 있거나 확장된 콘텐츠로 구성해야 한다. + 'article', 'aside', 'footer', 'header', 'nav' 요소의 자식 요소가 되면 안된다.
div(Division) 영역 : 웹사이트의 레이아웃을 만들 때 주로 사용
span 영역 : div와 같이 공간을 나타내는 태그. span은 컨텐츠의 길이만큼 width 영역이 잡힌다(content-width).
섹션 영역 : 본문(article)을 여러개 포함하고 있는 공간
아티클 영역 : 본문의 주 내용이 들어가는 공간
aside 영역 : 페이지의 다른 콘텐츠들과 약간의 연관성을 가지고 있지만, 해당 콘텐츠들로부터 분리시킬 수 있는 콘텐츠로 구성된 페이지 영역을 정의할 때 사용.
